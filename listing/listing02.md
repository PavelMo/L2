Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) { //в данном случае возвратное значение задаётся тут
	defer func() {
		x++ //возвратное значение изменяется
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x //возратное значение устанавливается как 1, а не как х
	        //и соответственно отложенная функция изменяет переменную х а не возвратное значение
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2  
1

defer запускает  отложенную функцию после return операторa, но до фактического возврата функции, что позволяет 
изменять возвращаемые результаты. Однако в данных функциях возратное значение устанавливается в 
разное время и соответствнно получаются разные результаты. 

Выражение defer добавляет вызов функции после ключевого слова defer в стеке приложения.
Все вызовы в стеке вызываются при возврате функции, в которой они добавлены. 
Поскольку вызовы помещаются в стек, они производятся в порядке от последнего к первому.
```
