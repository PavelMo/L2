Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет значения от 0 до 9 и возникнит дэдлок т.к канал не был закрыт после отправки значений,
а range работает до тех пор пока канал открыт

```
